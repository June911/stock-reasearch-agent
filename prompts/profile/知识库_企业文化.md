# 企业文化深度分析 Agent 框架

## 一、任务目标

对 **[公司名称]** 进行企业文化深度分析，回答三个核心问题：
1. 企业家可信吗？
2. 真实文化是什么？
3. 文化能支撑战略吗？

最终输出：投资视角的文化评估报告 + 风险清单 + 监控建议

---

## 二、概念定义

### 2.1 文化四象限（Cameron-Quinn模型）
```
              灵活性/自主
                   │
       创新文化    │    家族文化
      (Adhocracy)  │    (Clan)
                   │
  外部导向 ────────┼──────── 内部导向
                   │
       市场文化    │    层级文化
       (Market)    │   (Hierarchy)
                   │
              稳定性/控制
```

| 文化类型 | 核心特征 | 关键词信号 | 适合战略 |
|----------|----------|------------|----------|
| 创新文化 | 冒险、敏捷、颠覆 | innovation, risk-taking, disrupt, agile, experiment | 技术突破、颠覆式创新 |
| 市场文化 | 结果、竞争、客户 | results, winning, performance, customer, fast | 快速扩张、市场争夺 |
| 层级文化 | 流程、稳定、控制 | process, compliance, stability, control, efficiency | 成本领先、稳定运营 |
| 家族文化 | 协作、关怀、忠诚 | teamwork, caring, people-first, loyalty, family | 客户亲密、长期服务 |

### 2.2 评分标准

**Glassdoor评分**：
| 评分 | 判断 |
|------|------|
| >4.0 | ✅ 优秀 |
| 3.5-4.0 | ⚠️ 良好 |
| <3.5 | 🔴 需关注 |

**CEO认可度**：
| 比例 | 判断 |
|------|------|
| >85% | ✅ 优秀 |
| 70-85% | ⚠️ 良好 |
| <70% | 🔴 问题 |

**言行一致度**：
| 比例 | 判断 |
|------|------|
| >80% | ✅ 高度一致 |
| 60-80% | ⚠️ 部分一致 |
| <60% | 🔴 严重不一致 |

### 2.3 战略-文化匹配表

| 战略类型 | 最匹配 | 次匹配 | 不匹配 |
|----------|--------|--------|--------|
| 技术创新/颠覆 | 创新文化 | 市场文化 | 层级文化 |
| 快速扩张/抢市场 | 市场文化 | 创新文化 | 家族文化 |
| 成本领先/效率 | 层级文化 | 市场文化 | 创新文化 |
| 客户亲密/服务 | 家族文化 | 市场文化 | 层级文化 |

搜索5: [CEO名字] Form 4 insider trading stock sales
搜索6: [CEO名字] background biography career history

## 五、注意事项

1. **阶段依赖**
   - 阶段1→2→3→4→5 必须串行
   - 后阶段引用前阶段输出时，使用"来自阶段X"标注

2. **证据优先**
   - 每个判断必须有证据支撑
   - 区分"事实"和"推断"，推断需标注

3. **搜索策略**
   - 每个搜索词如果没有结果，尝试变体
   - 优先官方来源（10-K、Proxy、公司官网）
   - Glassdoor等第三方作为验证

4. **判断原则**
   - 敢于下判断，不要模棱两可
   - 矛盾点是最有价值的发现
   - 趋势比现状更重要

5. **输出规范**
   - 严格按模板输出，保证格式一致
   - 缺失数据填"未找到"，不要编造
   - 每阶段结束必须有"一句话结论"
