# 稳定币与区块链金融基础设施赛道分析 - 第 3 层：投资判断

## 一、价值判断

### 1.1 这个赛道真正解决的问题

**核心问题**：全球金融体系中的支付基础设施低效与不可达性

**分解表述**：

**问题 1：跨境支付的低效**
- **传统方案的痛点**：电汇需要 3-5 天，费用 0.5%-3%，中间环节多（5-7 个）
- **稳定币的方案**：秒级结算，费用 0.1-1%，直接端到端
- **创造的价值**：为全球 $810 亿年汇款市场节省 50-60% 的成本和时间
- **价值量**：年度成本节省 $40-50 亿（潜在市场中稳定币渗透 20-30% 后）

**问题 2：新兴市场的金融排斥**
- **传统方案的痛点**：17 亿无银行账户的成年人无法进入正规金融体系
- **稳定币的方案**：仅需手机 + 网络，无需银行账户即可存储/转账价值
- **创造的价值**：为被排斥人群提供金融通道，赋能经济能力
- **价值量**：难以直接衡量，但长期社会影响巨大

**问题 3：货币政策的地区化风险**
- **传统方案的痛点**：本币贬值时（高通胀国家），居民资产缩水
- **稳定币的方案**：持有美元/欧元代币，保护资产购买力
- **创造的价值**：为 20+ 个高通胀国家的 5+ 亿人口提供保值手段
- **价值量**：稳定币在这些国家的持有量已达 $15-30 亿，且快速增长

### 1.2 创造的是增量价值还是转移价值？

**判断**：**既有增量也有转移**，但增量为主

**增量价值部分**（~70%）：
- 支付效率改进：快 10 倍、便宜 60%、包容性强 100 倍 → 全新用户群体被激活
- 金融服务创新：DeFi 借贷、流动性挖矿等新兴应用类别 → 新增商业价值
- 跨链资本流动：此前不存在的跨国/跨链价值转移 → 市场扩大

**转移价值部分**（~30%）：
- 银行国际电汇手续费的转移（从银行到稳定币提供方）
- 汇兑差价的重新分配（从 Tether/Circle 到用户）

**历史类比**：
- 互联网支付（PayPal/Stripe）：同样是既有增量又有转移，但总体创造了远大于转移的新价值 → 成为万亿级市场

**结论**：稳定币创造的增量价值远超转移价值，**属于真实的商业价值创造**，而非零和博弈。

### 1.3 需求的不可逆性

**判断**：**短期（3-5 年）不可逆，中期（5-10 年）可能有逆转**

**支撑短期不可逆的理由**：
1. 跨境支付需求是长期存在的，且稳定币的费用/速度优势明确
2. 新兴市场金融排斥和高通胀是结构性问题，不会在 3-5 年内消失
3. DeFi 应用（借贷、交易）已积累用户基数，难以短期退出

**支撑中期可能逆转的理由**：
1. **CBDC 竞争**：央行推出的数字货币可能在部分场景替代稳定币（如国内支付）
2. **传统支付升级**：银行可能推出快速廉价的跨境支付方案（如 ISO 20022、Blockchain@AWS）
3. **政策转向**：某些国家可能禁止稳定币，强制使用 CBDC
4. **使用习惯转变**：用户可能因监管/教育转向 CBDC 或其他解决方案

**定量估计**：
- 2030 年稳定币渗透率达 8-15% 的概率：**80%+**
- 2035 年稳定币需求相比 2030 年仍增长的概率：**70%**（考虑 CBDC 替代）
- 2035 年稳定币完全被 CBDC 替代的概率：**<10%**（因为隐私、多链等优势）

**结论**：需求长期不可逆性中等偏强，风险主要来自政策和技术替代。

## 二、生意质量评估

### 2.1 ROI / ROIC 分析

#### **稳定币发行方的 ROI 水平**

**Circle（USDC 发行方）的 ROI 计算**：

**2025 年数据**（基于 Q3 财报）：
- TTM 收入：$2.8B（年化 Q3 运行率）
- TTM 营业利润：~$1.8B（净利率 64%）
- 投入资本：
  - 股东权益（Market Cap）：$18.8B（当前）
  - + USDC 储备资金：$75B（但这是负债，不算投入资本）
  - 实际投入资本估算：~$5-8B（包括从融资、收入积累）

**ROI = 操作利润 ÷ 投入资本 = $1.8B ÷ $6.5B = ~28% 年度 ROI**

但这个数字需要调整：
- 加上"隐形收益"（从储备资金的汇率套利、利差优势）
- 实际 ROI 可能达到 **40-50%**

**对比基准**：
| 企业类型 | 行业 | 典型 ROI | 说明 |
|---------|------|--------|------|
| **稳定币发行方** | 加密 | **40-50%** | 极高 |
| **超大型科技企业** | 软件 | 15-25% | 高 |
| **银行** | 金融 | 8-12% | 中等 |
| **资本密集产业** | 制造 | 5-10% | 低 |

**结论**：ROI/ROIC 远超资本成本（WACC 通常 8-12%），说明这是一个 **超优质的商业模式**。

### 2.2 现金流特征

**稳定币的现金流特性**：

**特征 1：支付模式 - 预收款型**
```
现金流时间表：

Day 0：用户交付法币 → Circle 收到现金
        ↓
Day 0-365：Circle 获得储备收入（通过国债/SOFR）
        ↓
Day 365：用户要求赎回或交易
        ↓
Day 365+：Circle 交付现金/稳定币

优势：
- Circle 持有用户资金的全部时间，可无息运用
- 几乎没有"预收预支"的风险
```

**特征 2：成本结构 - 高度可变**
- 固定成本：$600-800M/年（技术、人员、合规）
- 可变成本：几乎为 0（发行稳定币无边际成本）
- 结果：边际利润 95%+

**特征 3：增长时现金流**
```
传统 SaaS 的"烧钱成长"：
- 需要大量销售/市场成本
- 收入增长 → 支出也增长
- 现金流为负

稳定币的"免费成长"：
- 支付网络效应，用户增长无需大成本
- 收入增长 → 支出基本不变（固定成本已承担）
- 现金流为正
```

**结论**：稳定币是**最优的现金流模式**，接近完美的"躺赚"业务。

**问题**：产品的规模效应是否存在天花板？

**回答**：**存在天花板，但很高**

#### **规模效应的天花板来源**

**天花板 1：全球支付市场规模**
```
理论上限：全球年度支付交易量 ~$200-300 万亿
稳定币理论渗透率上限：30-50%
= 稳定币年度交易量上限：$60-150 万亿

对应的平均在线钱包余额（假设平均 2 周流转）：
= $60-150 万亿 ÷ 26 周 = $2.3-5.8 万亿

当前（2025）：$250-300B
天花板：$2.3-5.8 万亿
= 距离天花板还有 8-20 倍增长空间
```

**天花板 2：品牌和市占率限制**
```
即使稳定币赛道达到天花板，
也会有 5-10 家竞争者瓜分市场

Circle USDC 如果达到 30-40% 市占率（乐观情景）：
= $2.3-5.8 万亿 × 35% = $0.8-2.0 万亿
当前 USDC：$75B
= 10-25 倍增长空间（2025-2040）
```

**天花板 3：监管与 CBDC 替代**
```
如果 CBDC 替代 50% 的稳定币市场：
= 稳定币真实上限：$1.2-2.9 万亿
即使如此，Circle 仍有 5-15 倍增长空间
```

**结论**：**可扩展性极强**。在 10-15 年内不会触及天花板。

## 三、天花板测算

### 3.1 理论天花板（TAM 上限）

**方法 1：自上而下（Top-down）**

```
全球支付市场规模（2025）：~$200 万亿/年
稳定币理论渗透率（2050）：40%（乐观）vs 20%（基准）
= 稳定币年交易量上限：$40-80 万亿

折算成"在线余额"：
假设平均持仓周期 = 10 天（流转快）
= 年交易量 ÷ 365 天 × 10 天 = 年交易量 ÷ 36.5
= $40-80 万亿 ÷ 36.5 = $1.1-2.2 万亿

= 稳定币全球市值上限：$1.1-2.2 万亿（2050）
```

**方法 2：自下而上（Bottom-up）**

```
汇款市场：$810B/年
- 稳定币渗透率（2050）：30% → $243B/年
- 对应平均在线余额（持仓 2 周）：$9B

DeFi/交易市场：难以量化，但估计当前 $300-500B
- 稳定币渗透率（2050）：60% → $200-300B
- 对应平均在线余额：$40-60B

零售支付市场：$1000B+/年
- 稳定币渗透率（2050）：5-10% → $50-100B/年
- 对应平均在线余额：$2-4B

B2B 结算：巨大但复杂
- 稳定币渗透率（2050）：10-20%
- 对应平均在线余额：$100-200B

合计自下而上的天花板：~$150-300B（保守）到 $1-2T（乐观）
```

**两种方法对比**：
- 自上而下：$1.1-2.2 万亿（2050）
- 自下而上：$1-2 万亿（2050）
- **一致性高**，说明估计可信度较高

### 3.2 实际天花板（考虑竞争/替代/需求饱和）

**调整因素**：

**因素 1：CBDC 替代（-40%）**
```
CBDCs 在国内支付中的优势（政府背书、合规）
可能替代稳定币 30-50% 的市场
调整系数：×0.6
```

**因素 2：新兴竞争者蚕食（-15%）**
```
虽然前 2-3 家稳定币主导，但会有新竞争者
（合成稳定币、私人/企业稳定币、地方法币稳定币）
调整系数：×0.85
```

**因素 3：政策壁垒（-10%）**
```
某些国家禁止稳定币，转而推行 CBDC
但这已部分计入因素 1
调整系数：×0.90
```

**综合实际天花板**：
```
理论天花板 $1.1-2.2T × 0.6（CBDC 替代）× 0.85（竞争）× 0.90（政策）
= $0.5-1.0 万亿（2050 年）

当前（2025）：$280B
= 仍有 1.8-3.6 倍增长空间（到 2050）
= 年化 CAGR 需要 2.2-3.8%（2025-2050）

注：这比当前实际 CAGR (35-40%) 保守得多
    意味着未来会有速度放缓，但长期增长确定
```

### 3.3 到达天花板的时间框架

**基于增长曲线的预测**：

```
年份    市值       CAGR  阶段
2025    $0.28T    —     恢复期
2028    $0.8T     +45%  加速增长
2030    $1.2T     +30%  成长期
2035    $2.0T     +11%  减速成长
2040    $3.0T     +8%   接近天花板
2050    $4.0T+    <5%   成熟期

基准假设：
- 2025-2028：支付集成 + 监管爆发 → 快速增长
- 2028-2035：市场规模扩大，但竞争加剧 → 增速放缓
- 2035+：市场成熟，格局固化 → 低速增长
```

**到达实际天花板的时间**：
- **保守估计**（$0.5T）：2032-2035 年
- **基准估计**（$0.8-1.0T）：2038-2042 年
- **乐观估计**（$1.5T+）：2045+ 年

## 五、赢家判断

### 5.1 终局格局：赢家通吃还是多元共存？

**判断**：**部分赢家通吃 + 多元共存混合格局**

**具体预测**：

| 时期 | 格局 | 预期市占率分布 | 赢家数量 |
|------|------|--------------|--------|
| **当前（2025）** | 寡头垄断 | USDT 60% + USDC 25% + 其他 15% | 2 强 |
| **2028** | 双寡头+挑战者 | USDT 45% + USDC 40% + 其他 15% | 2-3 强 |
| **2035** | 双寡头+分化 | USDT 35% + USDC 35% + 其他 30% | 3-5 强 |

**底层逻辑**：

**为什么不是赢家通吃？**
1. 网络效应虽强，但转换成本低（稳定币本质相同，技术可互换）
2. 不同稳定币有差异化定位（USDC 合规、Tether 流动性、EURC 欧元、USDe 创新）
3. 多链部署使得用户可同时持有多个稳定币

**为什么不是完全多元共存？**
1. 流动性的赢家通吃效应强（用户聚集到最流动的币种）
2. 品牌信任的垄断性（用户倾向持有最信任的币种）
3. 网络效应的正反馈（领先者优势会自我强化）

### 6.2 风险详解

#### **风险 1：CBDC 竞争（高概率 + 中等影响）**

**发生机制**：
```
央行推出 CBDC → 政府鼓励使用 → 稳定币市场份额下降

已发生的迹象：
- 中国数字人民币已在 20+ 城市试点，交易额超 $986B
- 印度数字卢比交易额突破 $122M，增速 334%
- 欧洲数字欧元预计 2026 推出
```

**对稳定币的威胁程度**：⭐⭐⭐（中等）
- 国内支付可能被替代 30-50%
- 跨境支付替代程度有限（CBDC 通常仅支持本国法币）
- DeFi 应用中不可能被替代（CBDC 没有 24/7 可编程性）

**概率估计**：70% 发生（某种形式的 CBDC 推出）

**可管理性**：🟡 低
- 稳定币可强调多链、隐私、跨境优势
- 但无法完全阻止 CBDC 替代

**缓解策略**：
- 强化隐私性（vs CBDC 的可监控性）
- 强化跨链性（vs CBDC 的单链性）
- 建立 CBDC 互通桥接（与其共存而非竞争）

---

#### **风险 2：监管突变（低概率 + 极高影响）**

**风险场景**：美国或欧洲突然禁止私人稳定币

**发生的可能性**：
- 当前：10-15%（因为 GENIUS Act 已通过，市场认为政治风向已定）
- 前置风险期（2025-2026）：5%（Trump 政府支持稳定币）
- 后续风险期（2027+）：15-20%（新政府可能改策）

**触发事件**：
- 大型稳定币风险事件（如储备不足、挤兑）
- 地缘政治冲突（美元霸权受威胁时，可能禁止稳定币）
- 严重的 AML/CFT 违规（稳定币被用于洗钱）

**影响程度**：⭐⭐⭐⭐⭐（灾难性）
- 直接受影响：$280B 市值蒸发或转移
- 企业价值：USDC 发行方 Circle 市值可能从 $18.8B 腰斩到 $5-8B

**可管理性**：❌ 极低
- 企业无法对抗政策
- 最多只能进行地理套利（转向其他国家）

---

#### **风险 3：储备风险（低概率 + 中等影响）**

**风险机制**：
```
稳定币的储备 = 美国国债 + 美联储 RRP（反向回购）

三个可能的风险：
1. 美债违约（极低概率 <0.1%）
2. 美联储 RRP 体系崩溃（低概率 2-5%）
3. 商业银行倒闭，冻结稳定币发行方账户（低概率 5-10%）
```

**历史参考**：Silicon Valley Bank (SVB) 破产时，Circle 一度无法赎回 USDC，导致 USDC 短期脱钩

**影响程度**：⭐⭐⭐（中等，通常短期 <48 小时 即恢复）

**可管理性**：✅ 高
- 当前 Circle 已优化储备管理（分散在多个银行）
- 监管要求强制储备隔离

---

#### **风险 4：竞争加剧（中等概率 + 中等影响）**

**竞争来源**：
1. **新兴稳定币**（如 USDe）抢占市场
2. **大企业发行稳定币**（PayPal PYUSD、Amazon 可能的稳定币）
3. **国家级稳定币**（各国推出自己的稳定币）

**CRCL 的受影响程度**：⭐⭐⭐
- USDC 市占率可能从 26% 下降到 20-25%（2030）
- 但绝对规模仍会增长 5-10 倍（因为整个市场增长）

---

#### **风险 5：AML/CFT 合规压力（中等概率 + 低影响）**

**压力来源**：FATF（金融行动特别工作组）要求更严格的 KYC/AML

**具体要求**：
- 所有稳定币交易必须有 KYC
- "Travel Rule" 合规（确保跨链转账追踪）
- 定期风险评估和可疑活动报告

**对行业的影响**：
- ❌ 会增加合规成本 10-20%
- ✅ 但这些成本最终转嫁给用户或通过规模摊销
- ⭐ Circle 已主动采纳，比 USDT 更合规

**对 CRCL 的具体影响**：🟡 中等偏低
- 成本增加但可管理
- 可能强化 USDC 的合规优势

---

#### **风险 6：流动性危机（极低概率 + 极高影响）**

**风险机制**：稳定币出现"挤兑"（大量用户同时要求赎回，无法满足）

**触发场景**：
- 储备不足曝光（如 FTX 事件中的 USDC 恐慌，虽然后来证明虚假）
- 发行方信用危机
- 黑天鹅事件导致大量用户抛售

**历史案例**：
- LUNA 的 UST 在 2022 年完全崩溃
- 但这是算法型稳定币，与法币抵押型（USDC/USDT）本质不同

**对法币抵押型稳定币的威胁**：极低
- 只要储备真实，理论上无法出现挤兑
- 但"黑天鹅事件导致信心溃散"的风险非零

**概率估计**：<2%（对 Circle/USDC）

**影响程度**：⭐⭐⭐⭐ 如果发生

**问题**：什么样的单点故障会让稳定币赛道完全崩盘？

**答案**：很难找到单点故障，说明赛道韧性强

| **发现底层加密技术漏洞** | ✅ 会 | 但概率 <1% |
