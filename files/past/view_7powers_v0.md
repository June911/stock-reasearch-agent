#### Power 4: 转换成本（0-5分）

**Stage 1判断**：5分

**证据验证**：

**技术迁移成本分解**：
```
1. 代码重写成本
   - CUDA优化代码难以直接迁移到ROCm
   - 需要重写、测试、验证
   - 大模型项目：几万行代码，重写成本 > $1M/项目

2. 人才成本
   - CUDA开发者在市场上充足（供给多）
   - ROCm开发者稀缺（供给少）
   - 企业需要重新培训团队（6-12个月）
   - 成本：每人 $50k-100k（机会成本）

3. 系统集成成本
   - GPU集群一旦部署，架构难变
   - NVIDIA NVLink全家桶：GPU互联、系统软件都优化
   - 迁移到AMD需要改造数据中心架构
   - 停机时间对互联网公司极其贵（$1M+/小时）

4. 经济成本（已发生的）
   - 5-7年GPU集群的累积优化投入
   - CUDA库、驱动程序、模型优化已完成
   - 迁移=报废这些投资
```

**定量估计**（来自business_model分析）：
```
单一大客户迁移成本：
- 代码重写：$10-50M
- 人才培训：$5-10M
- 系统改造：$20-50M
- 停机损失：$50-100M
- 总计：$85-210M

NVIDIA H100价格下降空间：
- 现在$22k/片，可能降至$15k/片（32%）
- 降价额：$7k/片
- 1000片集群节省：$7M

结论：迁移成本 $85-210M >> 降价收益 $7M
因此粘性极强
```

**验证：为什么MI300未能获得市场份额**：
```
2024年MI300推出：
- 价格：$18.5k（便宜23%）
- 性能：部分指标优于H100
- 但市场份额：<3%

原因剖析(layer2中的定量分析)：
采购决策 = 硬件性能(30%) + 生态(40%) + 价格(30%)
- H100评分：95+95+30 = 96.5
- MI300评分：100+60+35 = 89

即使便宜40%，生态差距导致总体评分更低
转换成本成为决定性因素
```

**评分**：**5分** ✓
- 转换成本极高（$85-210M/企业）
- 客户粘性超强（MI300失败案例印证）
- 定价权超强（即使价格便宜仍难获市场）

**长期可持续性**：★★★★★
- CUDA使用年限越长，历史投入越大，迁移成本越高
- 非线性强化：使用时间越久，锁定强度越强
